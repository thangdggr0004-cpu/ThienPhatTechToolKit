## 🛠️ CHI TIẾT CÁC TÍNH NĂNG CHUYÊN SÂU

### 1. 🛡️ Quét & Xử Lý Bản Quyền (Activation Scanner)
Công cụ bóc tách tình trạng bản quyền Windows/Office với thuật toán quét sâu 8 bước:
- **Phân tích chữ ký số phần cứng (HWID):** Nhận diện hệ thống dùng Key chung (Generic Key) để ép kích hoạt kỹ thuật số.
- **Phân tích Kênh cấp phép:** Phân loại giấy phép (Retail, OEM, Volume_MAK).
- **Truy vết máy chủ KMS:** Kiểm tra hệ thống có bị trỏ về các máy chủ ảo KMS giả mạo hay không.
- **Quét tệp tin & DLL bẻ khóa:** Tìm kiếm tàn dư của AutoKMS, KMSPico, SECOH-QAD, KMS38.
- **Kiểm tra Task Scheduler & Services:** Phát hiện các tác vụ tự động gia hạn bản quyền ngầm.
- **Tiêu diệt KMS / HWID 1-Click:** Gỡ Product Key, dọn dẹp Registry, clear ClipSVC tokens và trả máy về trạng thái bản quyền gốc.

### 2. 🖨️ Tiện Ích Máy In (Printer Utilities)
- **Quản lý máy in:** Xem danh sách, cổng (Port), trạng thái thực tế (Idle, Offline, Error) và đặt máy in mặc định.
- **Xem hàng đợi (Print Queue):** Theo dõi danh sách lệnh in đang chờ trực quan.
- **Sửa lỗi kẹt lệnh in 1-Click:** Khởi động lại Print Spooler và xóa sạch hàng đợi bị kẹt cứng lập tức.
- **Fix Share Máy In LAN:** Tự động nạp patch Registry để vá lỗi chia sẻ máy in mạng LAN kinh điển (`0x0000011b`, `0x00000709`).
- **In Trang Test & Driver:** In thử trang test, mở nhanh Device Manager hoặc gọi hộp thoại Add Printer mặc định của Windows.

### 3. 💻 Kiểm Tra Laptop & Thiết Bị (Laptop Tester)
- **Màn hình (Screen Test):** Quét 6 màu đơn sắc chuẩn (Trắng, Đen, Đỏ, Xanh lá, Xanh dương, Vàng) để phát hiện điểm chết (Dead Pixels) và hở sáng.
- **Bàn phím (Keyboard Tester - Full Screen):** 
  - Giao diện **Toàn Màn Hình** căn giữa cực chuẩn.
  - Hỗ trợ **3 loại Bố cục bàn phím**: Laptop 75%, Bàn phím đầy đủ 100% (có cụm Numpad bên phải) và Macbook Layout.
  - Tích hợp đầy đủ hàng phím `F1-F12`, `Esc`, `PrtSc`, `ScrLk`, `Pause`, `Ins`, `Del`, `Home`, `End`, `PgUp`, `PgDn`.
  - Bộ đếm số phím đã test + Nút **[🔄 Reset]** kiểm tra lại từ đầu.
- **Cảm ứng (Touch Screen Tester):** Thiết kế ma trận vẽ cảm ứng 4 góc giúp phát hiện liệt điểm cảm ứng cực kỳ chính xác.
- **Webcam & Microphone:** Kiểm tra độ nét Camera và âm thanh đầu vào Micro trực tiếp.
- **Sức khỏe Pin & Ổ cứng:** Đọc thông số thiết kế/dung lượng còn lại của Pin và cảm biến nhiệt độ thực tế SMART (°C) của ổ SSD/HDD.

### 4. ⚙️ Tối Ưu Nâng Cao & Tinh Chỉnh Windows (Windows Settings)
- **Thiết lập nhanh:** Bật Menu chuột phải cổ điển Windows 10 trên Win 11, hiển thị biểu tượng This PC, bật Photo Viewer cổ điển, chuyển Dark/Light Mode.
- **Tối ưu Dịch Vụ & Power Plan:** Chuyển hệ thống sang chế độ High Performance / Ultimate Performance.
- **⚡ Tối Ưu Nâng Cao (Extreme Performance Modal):**
  1. Tự động tạo **Restore Point** khôi phục an toàn trước khi tối ưu.
  2. Tắt HPET & Dynamic Tick giúp giảm độ trễ đầu vào (Input Lag).
  3. Tắt Network Throttling Index tăng tốc mạng khi chơi game.
  4. Giải phóng bộ nhớ đệm RAM (Purge Standby RAM).
  5. Tắt ứng dụng UWP chạy ngầm (Background Apps).
  6. Tắt Delivery Optimization (Windows P2P Update Download).
  7. Kích hoạt Windows Game Mode tối đa FPS.
  8. Tắt Startup Delay (Mở app tự khởi động lập tức khi vừa vào Win).
- **🔄 Nút 1-Click Undo (Trả về mặc định Windows):** Khôi phục toàn bộ cài đặt hệ thống quay về trạng thái gốc của Microsoft nếu không muốn tối ưu nữa.

### 5. 🧹 Dọn Dẹp Rác & Phục Hồi Windows (Junk Cleaner & Fixer)
- **Dọn rác đĩa C:** Quét an toàn thư mục Temp, Windows Update Cache, Icon Cache và Prefetch giải phóng hàng GB dung lượng đĩa.
- **Sửa lỗi file hệ thống:** Tích hợp bộ lệnh `SFC /scannow` và `DISM /RestoreHealth` khôi phục file hệ thống bị hỏng.
- **Khôi phục Windows Update:** Reset dịch vụ BITS, wuauserv và dọn dẹp thư mục SoftwareDistribution.

### 6. 🔓 Giải Mã BitLocker (BitLocker Manager)
- Dò tìm toàn bộ phân vùng bị khóa mã hóa BitLocker và thực hiện lệnh giải mã (Decrypt) hàng loạt bằng 1-Click để cứu hộ dữ liệu.

### 7. 💾 Sao Lưu Wi-Fi & Driver (Backup Manager)
- **Sao lưu Wi-Fi:** Trích xuất toàn bộ tên và mật khẩu Wi-Fi đã lưu trên máy ra file Text.
- **Sao lưu Driver:** Trích xuất toàn bộ Driver hiện tại (dạng .INF) ra thư mục dự phòng để chuẩn bị cài lại Win.

### 8. 🌐 Quản Lý Mạng & DNS (Network Config)
- Đổi nhanh DNS sang Google (`8.8.8.8`), Cloudflare (`1.1.1.1`), OpenDNS, Quad9.
- Đo độ trễ Ping tới các máy chủ lớn và Reset Adapter Card Mạng khi bị mất kết nối.

### 9. 📝 Chuẩn Hóa Office 1-Click (Office Standardizer)
- Tự động chuẩn hóa lề trang, font chữ chuẩn (Times New Roman / Calibri, size 14, giãn dòng 1.5pt) cho Word/Excel và dọn dẹp macro rác.

### 10. 🔄 Cập Nhật OTA Tự Động & Xóa File Cũ (Auto Updater)
- Tự động kết nối với GitHub API để phát hiện bản cập nhật mới.
- Tự động tải bản mới, chạy script cài đặt ngầm và **tự động xóa file `.exe` phiên bản cũ** trên đĩa.

## ⚡ ĐẶC ĐIỂM KỸ THUẬT VƯỢT TRỘI

- **🚀 Tốc độ bật ứng dụng cực nhanh (1 Giây):** Nhờ cơ chế `unpackDirName` kết hợp V8 Bytecode Caching, ứng dụng nẩy lên màn hình tức thì trong 1s mà không bị trễ giải nén.
- **🔋 Tối ưu CPU 0% khi thu nhỏ:** Tự động đóng băng luồng đo đạc khi thu nhỏ app xuống Taskbar.
- **🎨 Giao diện GPU 60 FPS:** Xây dựng trên React 19, Vite, Tailwind CSS & Electron với hiệu ứng chuyển trang mượt như nhung.
- **🔒 Ký chứng chỉ số (Digital Signature):** File exe được ký chứng chỉ mã hóa an toàn, không bị Windows Defender nhận nhầm là phần mềm độc hại.
